# 🗺️ Product Roadmap

## 📝 计划中 (Planned)

- [ ] **实现 custom 格式反向导入**
  - [ ] 基于用户自定义字段映射解析标注文件，转换为内部 `AnnotationExport`
  - [ ] 提供导入前字段映射预览与样例校验，避免错误映射静默污染标注数据
  - [ ] 导入后复用项目专用模板机制，标签快照不污染全局模板

- [ ] **实现多边形标注**
  - [ ] 支持连续点击添加多边形顶点，双击或按 Enter 完成绘制
  - [ ] 已完成的多边形可选中，顶点可单独拖拽调整
  - [ ] 更新 `coco`/`custom` 导出逻辑以支持多边形数据（VOC/YOLO 视格式本身是否支持决定是否跳过）

- [ ] **实现关键点标注**
  - [ ] 支持在图片上单击放置关键点，绑定标签
  - [ ] 关键点可选中并拖拽调整位置
  - [ ] 若需要关键点分组（如人体骨架点位关联），设计对应的数据结构扩展

- [ ] **实现图形类型工具栏切换与对应快捷键**
  - [ ] 工具栏增加图形类型切换按钮（矩形/多边形/关键点）
  - [ ] 绑定快捷键快速切换绘制模式

- [ ] **扩展标签配置以支持指定图形类型**
  - [ ] `LabelConfig.shapeType` 生效：不同标签仅能用于对应图形类型（或允许通用，视产品决策）
  - [ ] 标签配置面板 UI 相应调整，创建标签时可选择适用图形类型

- [ ] **实现视频文件导入与抽帧**
  - [ ] Rust 端接入视频处理能力（ffmpeg sidecar 或对应 crate），实现按固定间隔抽帧
  - [ ] 前端展示抽帧后的帧列表（类比此前的图片列表加载逻辑）

- [ ] **实现视频时间轴 UI**
  - [ ] 新建时间轴组件，展示当前视频总帧数、当前帧位置
  - [ ] 支持拖动时间轴快速跳转到指定帧

- [ ] **复用图片标注画布逻辑标注视频单帧**
  - [ ] 确认 `AnnotationShape.frameIndex` 字段正确关联每一帧的标注数据
  - [ ] 验证切换帧时标注加载/保存逻辑与图片标注一致

- [ ] **实现跨帧标注插值（可选，视需求优先级）**
  - [ ] 设计关键帧标注 + 中间帧线性插值的数据结构
  - [ ] 实现插值计算逻辑与对应 UI（标记关键帧、查看插值结果）

- [ ] **扩展导出格式以支持视频标注结果**
  - [ ] 评估现有导出格式（COCO/VOC/YOLO）是否有视频标注扩展惯例，或设计自定义结构
  - [ ] 实现对应导出逻辑并验证
