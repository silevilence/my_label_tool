# Change Log

## V0.3.0

### ✨ 新功能

- 应用启动后自动检查更新，也可在菜单中手动「检查更新」；发现新版本时显示当前版本、新版本与发布说明，确认后自动下载安装并显示进度，安装完成自动重启
- 绘制标注框时鼠标显示十字光标，并显示随鼠标移动的水平/垂直辅助线，方便精确对齐定位；两者均可在设置中单独开关
- 右上角标签快捷键列表会高亮当前新建标注框将使用的标签，便于快速确认默认绘制标签

## V0.2.1

### ✨ 新功能

- 按 `Ctrl+S` 可快速保存当前项目，无需再通过导出面板操作
- 画布默认模式下可显示标签快捷键速查面板（如 `1：人`、`Q：车`），方便记忆各标签对应的按键
- 设置中新增画布提示开关，可分别控制是否显示模式操作提示与标签快捷键速查
- 设置中新增标签名显示配置，可分别控制默认、选择、标注三种模式下是否在标注框上显示标签名
- 切换图片时后台预加载下一张图片，再次切换时可直接复用，减少空白等待

### 🚀 优化

- 连续打标时，画完一个框不再自动选中它，此时按标签快捷键只切换后续新框的默认标签，避免误改刚画完的框；主动选中已有框后仍可按快捷键修改其标签
- 切换默认标签时，画布上方会短暂显示当前标签名与颜色，方便快速确认
- 切换大图时显示加载进度反馈，不再像卡住一样无响应
- 保存项目时显示进度条与「保存完成」提示，保存状态更明确
- 模式操作提示补充了 `Shift` / `Ctrl` 修饰键、`Ctrl+S` 保存等说明，上手更轻松

## V0.2.0

### ✨ 新功能

- 从外部 YOLO 数据集创建项目：手上只有 YOLO 格式的图片和标注目录时，无需本工具项目文件即可直接导入并生成项目；自动读取 `classes.txt` 生成标签（无则按类别序号命名，如 `class_0`），导入后保存项目配置，再次打开该目录自动按本项目加载；导入完成后给出缺失图片、缺失标注与异常标注行的数量摘要
- 一键跳转下一张 / 上一张未标注图片，快速定位待处理内容
- 标注进度统计：实时显示已标注、未标注与总图片数量，并附完成进度条

### 🚀 优化

- 标注框缩放支持自由调整宽高比，不再锁定比例，可独立改变宽和高
- 标签与模板管理整合为弹窗，主界面更清爽

## V0.1.0

### ✨ 新功能

**图片浏览**

- 打开本地图片文件夹即可开始标注，支持 JPG / PNG / BMP 格式，自动跳过损坏或空文件并给出提示
- 方向键快速切换上一张 / 下一张图片
- 滚轮以光标位置为中心缩放画布，Ctrl + 滚轮微调；Ctrl + 右键或中键拖动平移画布

**矩形框标注**

- 在图片上绘制矩形标注框，支持选中、拖拽移动、四角缩放调整
- 按住 Shift 强制进入选择模式，按住 Ctrl 强制进入标注模式，无需来回切换工具
- 针对大图片优化了标注与渲染流畅度

**标签体系**

- 内置「通用目标检测」与「道路交通」两套标签模板，开箱即用
- 自由增删改标签：名称、颜色、单键快捷键
- 标签模板管理：新建、保存、另存为、删除自定义模板（内置模板不可删除）
- 按下标签快捷键（如 1 / 2 / 3）快速切换当前标签，已有选中框同步切换其标签

**导入与导出**

- 导出标注：原生 JSON、COCO JSON、VOC XML、YOLO TXT、自定义字段映射 JSON 五种格式
- 导入标注：支持原生 JSON、COCO、VOC、YOLO 标注文件
- 打开图片文件夹时若检测到项目配置文件，提示自动加载

**操作体验**

- 撤销 / 重做：标注的新增、删除、移动、调整均可回退（默认 Ctrl + Z / Ctrl + Y）
- 选中标注框按 Delete 删除；提供「清空当前图片所有标注」按钮，需二次确认
- 通用快捷键配置面板：切图、缩放等快捷键可自定义并持久化保存

**应用特性**

- 离线运行、无需联网、无需安装额外依赖的 Windows 桌面应用
